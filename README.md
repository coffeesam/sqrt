#Simple QR Toolkit

##Usage
```
usage: ./sqrt text outpath [--auto] [--quiet]

  text       : text to encode in QR code
  output     : output path to write PNG image to

  OPTIONS

    --auto   : proceed to generate QR code without prompt
    --quiet  : suppress sending messages to stdout

```
